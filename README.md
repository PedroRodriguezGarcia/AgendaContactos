# Agenda de Contactos

## Opción elegida
Agenda de Contactos (Contact Book)

## Funcionalidades implementadas
- Añadir contactos con nombre, teléfono, email y etiquetas
- Validación de campos obligatorios y formato de email
- Listado dinámico de contactos
- Editar contactos existentes
- Borrar contactos
- Búsqueda en tiempo real por nombre, email o etiquetas
- Persistencia de datos usando localStorage

## Instrucciones de uso
1. Abrir el archivo `index.html` en un navegador.
2. Rellenar el formulario con los datos del contacto y hacer clic en "Añadir contacto".
3. El contacto aparecerá en la lista de abajo.
4. Para editar un contacto, hacer clic en el botón "Editar", modificar los datos en el formulario y añadir de nuevo.
5. Para eliminar un contacto, hacer clic en el botón "Borrar".
6. Usar el campo de búsqueda para filtrar contactos por nombre, email o etiquetas.
7. Los datos se guardan automáticamente en el navegador y se mantendrán al recargar la página.

## Estructura de archivos
- `index.html` → HTML principal
- `style.css` → Estilos CSS
- `app.js` → Lógica de la aplicación en JavaScript
